# HTML5 Snake

HTML5 Snake game written in ES6/CreateJS using webpack (currently in progress).
Requirements:
------
+ Node & npm
+ Bower
+ Webpack & Webpack-dev-server

Installation:
------
To get started clone this repository and type:

``
$ npm install
``

``
$ bower install
``

To start webpack-dev-server type:

``
$ npm start
``
